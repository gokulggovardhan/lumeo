# Lumeo roadmap

## Current development: Non-AI studio features

### Available now / build first

- Premium studio layout
- Auto-save editor settings
- Direct export using latest selected settings
- Export quality presets:
  - 720p · 30fps
  - 1080p · 30fps
- Aspect ratios:
  - 9:16 Vertical
  - 1:1 Square
  - 4:5 Portrait
  - 16:9 Landscape
- Frame modes:
  - Full Frame
  - Original View
  - Blurred Background
- Basic trim/cut controls
- Output target summary
- Clean export status and download
