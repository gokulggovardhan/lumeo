# Lumeo

**Lumeo** is a premium online creative studio for creators to turn raw clips into clean, polished short videos.

Live website: https://lumeo.in

## What Lumeo does

Lumeo is designed for focused short-form video workflows. It helps creators upload a clip, trim it, reframe it, add titles, adjust sound, and export a clean MP4 video.

## Current features

* Premium public landing page
* Creator-focused Studio workspace
* Video upload flow
* Project-based editing workflow
* Reframe Studio for composition and safe framing
* Titles Studio with premium title presets
* Canvas formats for short-form and social outputs
* Export workflow for polished MP4 downloads
* Project delete and cleanup flow
* SEO foundation with sitemap, robots.txt, structured data, and Google indexing

## Product direction

Lumeo is being built as a clean, premium, browser-based creative studio for video creators, short-form content makers, educators, podcasters, developers, social media teams, and businesses.

The goal is not to copy large editing suites, but to create a focused creator workflow that feels simple, polished, and reliable.
